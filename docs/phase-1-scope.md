# Phase 1 Scope

Phase 1 delivers the independent IONOS swarm runtime only.

## Included

- One centralized runtime for combat and pro-wrestling
- MongoDB-backed jobs and artifacts
- Workers, retries, dead-letter records, audit-style agent runs
- Draft content, SEO reports, social drafts, data candidates, and pro-wrestling advisory artifacts
- Deployment files for IONOS using Docker Compose

## Excluded until Phase 2/3

- Direct publishing to the production Blog collection
- Direct match/contest creation in the production backend
- Wallet, payout, prediction, entry, or settlement writes
- Frontend dashboard integration
- Real social platform posting

## Production rule

The current website and backend are authoritative. The swarm produces artifacts. The backend decides what becomes live data.
